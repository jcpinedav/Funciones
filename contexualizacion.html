Contexualización

Introducción


El software básicamente contiene tres partes: una en donde se presentan los conceptos de relaciones matematicas en cuyo final se encuentra un ejercicio para parcticar sobre la identificación de tipos de funciones, la segunda muestra un ejercicio trabajado sobre los dominios de las funciones y para finalizar, una con ejercicios para trabajar sobre el rago de una función.




Objetivo


Formar conceptualmente acerca de ls funciones matematicas y el desarrollo de conocimiento procedimental para la resolución de problemas que involucren estos conceptos




Metodología


Se utiliza para el aprendizaje de conceptos de relaciones matematicas, comienza dando definición del concepto; posteriormente, se presentan ejemplos trabajados en la parte izquierda. En la parte derecha se da un ejercicio para que e efectúe paso a paso de acuerdo con el ejemplo de la izquierda. Siempre que el estudiante digite una una información y después de presionar enter, el programa provee retroalimentación sobre el resultado de las respuestas dadas. Aunque haya realizado los ejecicios bien, es recomendable hacer muchos de estos, para poder aprender. El hacer un ejercicio bien indica que comprendió, para realmente aprender se debe efectuar una gran cantidad de éstos
